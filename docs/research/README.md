## 核心概念速查

**斯特拉德拉左手六行（由内到外）：**
```
行1 对位低音（根音大三度上方的单音）
行2 根音（单音低音）
行3 大三和弦
行4 小三和弦
行5 属七和弦
行6 减七和弦
```

**列排列规律：** 相邻列相差纯五度（五度圈顺序）

**纽扣手风琴行内规律：** 相邻按钮相差小三度（3 个半音）
**纽扣手风琴行间规律：** 相邻行相差半音（1 个半音）
