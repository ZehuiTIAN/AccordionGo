# 斯特拉德拉低音系统（Stradella Bass System）

全球最广泛使用的手风琴左手键盘系统，几乎所有键盘手风琴和大多数纽扣手风琴的左手均采用此布局。

---

## 总体结构

**标准 120 低音配置：**
- 总按钮数：120 个
- 排列：**6 行 × 20 列**
- 每列对应一个音名（一个调性家族）
- 列与列之间按**五度圈**顺序排列

---

## 六行排列

从演奏者角度，**由内（靠近身体）向外**依次为：

```
内侧（靠近演奏者）
  行 1 ── 对位低音行（Counterbass）  ── 单音：根音的大三度上方
  行 2 ── 根音行（Bass）             ── 单音：基础低音根音
  行 3 ── 大三和弦行（Major）        ── 预置大三和弦
  行 4 ── 小三和弦行（Minor）        ── 预置小三和弦
  行 5 ── 属七和弦行（Dom 7th）      ── 预置属七和弦
  行 6 ── 减七和弦行（Dim 7th）      ── 预置减七和弦
外侧（远离演奏者）
```

> **触觉标记：** 标准 120 贝斯有三个触感参考按钮，均匀分布在五度圈上（相互相差 4 个纯五度）：
> - **C 列**：主标记，凸起点或小孔，最明显
> - **Ab 列**：次标记（C 向降号方向第 4 列）
> - **E 列**：次标记（C 向升号方向第 4 列）
>
> 这三个音在五度圈上等距，将整个面板三等分，演奏时不看键盘可凭触感快速定位。
>
> **内侧低、外侧高：** 六行在面板上呈斜向排列——靠近风箱侧（内侧：对位低音行、根音行）的按钮位置偏低（靠近腹部/音窗方向），远离风箱侧（外侧：减七行等）的按钮位置偏高（靠近肩部/背带方向）。这一斜向排列方便演奏者左手自然弯曲定位。

---

## 根音行（Bass Row）——五度圈排列

20 行从上到下（演奏者视角，上 = 靠近肩部；**最上行 Bb/A#，最下行 A**）按升四度方向排列：

**外区 12 行（低音寄存器，C 以上各行用升号）：**
```
A# – D# – G# – C# – F# – B – E – A – D – G – C – F
```

**内区 8 行（高音寄存器，C 以下各行用降号）：**
```
Bb – Eb – Ab – Db – Gb – B – E – A
```

规律：**从上往下每行升高纯四度（5 半音）**，等价于从下往上每行升高纯五度（7 半音）。
- C 在外区第 11 行（靠下端），是触感标记的参考点，也是升/降号分界

> **记谱惯例**：以 C 行为分界——C 以上（外区）的变化音用升号（A#/D#/G#/C#/F#）；C 以下（内区）的变化音用降号（Bb/Eb/Ab/Db/Gb）。B/E/A/D/G/C/F 为还原号，两区相同。

## 对位低音行（Counterbass Row）

每个按钮发出的音是**同列根音向上一个大三度**的音：

| 根音列 | 对位低音 |
|--------|---------|
| C | E |
| G | B |
| D | F# |
| A | C# |
| E | G# |
| B | D# / Eb |
| F | A |
| Bb | D |
| Eb | G |
| Ab | C |
| Db | F |
| F# | A# / Bb |

常见用途：
- 构建转位低音线条（如 C/E 第一转位效果）
- 增加低音声部的旋律性
- 与根音配合形成八度或其他音程

## 和弦行详解

每个和弦行有 20 个按钮，按下即触发**预置和弦**（同时发出和弦所有音）。和弦排列顺序同根音行（五度圈）。

### 大三和弦行（Major Chord Row）

| 按钮列 | 发出和弦 | 组成音 |
|--------|---------|-------|
| C | C 大三 | C – E – G |
| G | G 大三 | G – B – D |
| D | D 大三 | D – F# – A |
| … | … | … |

### 小三和弦行（Minor Chord Row）

| 按钮列 | 发出和弦 | 组成音 |
|--------|---------|-------|
| C | C 小三 | C – Eb – G |
| G | G 小三 | G – Bb – D |
| D | D 小三 | D – F – A |
| … | … | … |

### 属七和弦行（Dominant 7th Row）

| 按钮列 | 发出和弦 | 组成音 |
|--------|---------|-------|
| C | C7 | C – E – G – Bb |
| G | G7 | G – B – D – F |
| D | D7 | D – F# – A – C |
| … | … | … |

### 减七和弦行（Diminished 7th Row）

| 按钮列 | 发出和弦 | 组成音 |
|--------|---------|-------|
| C | C dim7 | C – Eb – Gb – Bbb(A) |
| G | G dim7 | G – Bb – Db – Fb(E) |
| … | … | … |

> 减七和弦具有对称性（每小三度等价），12 个位置实际只有 3 种不同音响，每种出现 4 次。

---

## 五度圈与布局的关系

```
五度圈（向右 = 升高纯五度）：
… F – C – G – D – A – E – B – F# – Db – Ab – Eb – Bb – F …
```

| 操作 | 音乐含义 |
|------|---------|
| 同一行，相邻列 | 相差纯五度 |
| 同一列，不同行 | 同音名不同和弦类型 |
| I–V–I 进行 | 在相邻两列间来回 |
| 关系大小调 | 根音行与小三和弦行**同一列**即是关系大调/小调 |
| 完美终止式 | 属七行某列 → 根音行同列（V7 → I） |

## 常用左手伴奏型

### 华尔兹（3/4 拍）— Oom-pah-pah

```
拍 1：根音行（低音）
拍 2：和弦行
拍 3：和弦行
```

### 进行曲 / 波尔卡（2/4 拍）— Oom-pah

```
拍 1：根音行（低音）
拍 2：和弦行
```

### 探戈（4/4 拍）

```
拍 1：根音
拍 2 前半：和弦
拍 2 后半：根音
拍 3：空（休止）
拍 4：和弦
```
