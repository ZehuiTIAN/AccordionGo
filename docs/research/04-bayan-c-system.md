# 巴扬手风琴 C 系统（C-System / French System）

C 系统是半音阶纽扣手风琴（CBA）右手键盘的另一主要变体，与 B 系统互为镜像。

---

## 地理分布

主要流行地区：法国、德国、比利时、荷兰、瑞士、意大利等西欧国家，以及中国部分地区。

---

## 右手键盘结构

### 总体布局

C 系统同样有 **5 行主要按钮 + 2 行辅助行**，共 7 行。

从演奏者视角，**从右（外侧）到左（内侧）**：

```
行 7（最内） ── 辅助行，重复行 5 的音
行 6         ── 辅助行，重复行 4 的音
─────────────────────────────────
行 5         ── 主要行
行 4         ── 主要行
行 3         ── 主要行
行 2         ── 主要行
行 1（最外） ── 主要行（C 位于此行）★
```

> ★ **中央 C** 位于**第 1 行**（最外侧）。

### 单行内音程关系

- 同一行内，**相邻按钮之间相差小三度（3 个半音）**——与 B 系统相同

### 相邻行之间

- 相邻两行之间，相差**半音（1 个半音）**——与 B 系统相同

### 按钮倾斜方向

C 系统按钮视觉上呈 **\ 方向**（反斜线，从左上到右下）排列。
这与 B 系统（/ 正斜线）恰好**镜像对称**。

---

## B 系统 vs C 系统详细对比

| 对比项 | B 系统 | C 系统 |
|--------|--------|--------|
| 中央 C 位置 | 第 3 行（从右数）| 第 1 行（最外侧）|
| 按钮倾斜方向 | /（正斜线）| \（反斜线）|
| C 大调音阶起始行 | 中间行 | 最外侧行 |
| 主要流行地区 | 东欧、俄罗斯、挪威 | 西欧、法国、中国 |
| 代表乐器称呼 | Bayan（巴扬）| 法式纽扣手风琴 |
| 相互转换 | 不可直接转换，需重新学习指法 | — |

---

## C 大调音阶的 C 系统指法

C 系统中，C 大调音阶指法示例（从 C4 开始上行）：

```
C  ── 行1，位置 k
D  ── 行2，位置 k
E  ── 行3，位置 k
F  ── 行1，位置 k+1
G  ── 行2，位置 k+1
A  ── 行3，位置 k+1
B  ── 行1，位置 k+2（或 行4 k+1）
C  ── 行1，位置 k+2（高八度）
```

同样地，**所有调式的音阶和和弦指法模式完全相同**，转调只需平移手指起始位置，无需改变指法形状。

---

## 共同优势（B 系统和 C 系统）

相比键盘手风琴，纽扣手风琴的最大优势：

1. **转调无需改变指法形状** — 所有大调音阶指法相同，只需平移
2. **更宽音域** — 在相同体积内可容纳更多按钮，通常 4–5 个八度
3. **更适合古典复调音乐** — 大音域 + 自由低音系统
4. **手指移动距离更短** — 按钮比钢琴键更紧凑，适合快速跑动
