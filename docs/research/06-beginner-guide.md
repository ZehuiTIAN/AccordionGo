# 手风琴入门指南

---

## 一、坐姿（Sitting Posture）

### 椅子选择

- 选**无扶手的硬质椅子**
- 高度：双脚平放地面，膝盖弯曲约 90°
- 避免沙发或软椅（影响腰背支撑，导致风箱控制不稳）

### 身体姿势

- 腰背**挺直但放松**，不僵硬，也不弓背
- 肩膀自然下沉，不耸肩
- 双脚平放地面；左脚可略微向前以稳定重心

### 手风琴位置

- 乐器放于**左腿上**（右腿辅助夹住）
- 键盘应位于胸前，大约在下颌附近高度
- 乐器整体**居中于躯干**，不偏左或偏右

---

## 二、持琴方式（How to Hold）

### 背带调整（核心）

- **右侧背带**应比左侧**略长**，为右手上下滑动提供活动空间
- **左侧背带**应较紧，固定乐器，使左手控制风箱时乐器稳定

### 左手姿势

1. 将左手穿入左侧皮质手腕带
2. **拇指不按键**——拇指抵在低音格栅侧面，起支撑和引导风箱方向的作用
3. 四指（食指到小指）略弯曲，用**指尖肉垫**按按钮
4. 手腕保持自然，不过度弯曲

### 右手姿势（键盘手风琴）

- 前臂保持**水平**
- 手掌自然拱起，手指接近 90° 竖立触键
- 用指尖肉垫触键，不用指甲或关节侧面触键

---

## 三、风箱技巧（Bellows Technique）

### 基本原理

风箱（Bellows）驱动空气流过金属簧片发声。
- **推压（Push / 合风箱）**：向内合拢，音量由合拢速度决定
- **拉张（Pull / 开风箱）**：向外展开，同样可以发声

> 与口琴不同，手风琴推拉风箱**音高不变**，只影响音量和音色。

### 风箱动作要领

- 标准动作为**扇形运动**：风箱上部开合幅度大，下部幅度小，形成扇形
- 推压时：左手微微向身体方向用力，下部保持稳定
- 拉张时：左手向外，以手腕带为支点均匀用力

### 换气（Bellows Change / 换向）

- 在乐句**自然停顿**或**休止符**处换向
- 避免在乐句进行中突然换向（会产生明显断音）
- 练习方法：先单独练习匀速推拉，再逐渐加入按键

### 常见错误

| 问题 | 原因 | 纠正方法 |
|------|------|---------|
| 风箱抖动 / 颤动 | 气流不稳定 | 练习匀速慢速推拉 |
| 换气时声音断裂 | 换向太突然 | 在换向瞬间短暂加快按键压力 |
| 风箱开合过大 | 浪费气流 | 控制在实际需要的幅度内 |
| 音量无法控制 | 不理解风箱与音量的关系 | 专项练习：只靠风箱速度控制响弱 |

### 音量控制原理

```
风箱速度快 → 气流强 → 音量大（Forte）
风箱速度慢 → 气流弱 → 音量小（Piano）
```

这是手风琴独特的动态控制方式，与钢琴（靠触键力度）完全不同。

## 四、双手协调（Left-Right Coordination）

### 学习顺序

1. **分手练习**：先右手旋律熟练，再左手伴奏型熟练
2. **节奏统一**：不拿手风琴，在桌上分别拍打左右手节奏，确认节拍准确
3. **合手练习**：极慢速度（原速的 1/4）开始，右手旋律音与左手低音按下时机精确对齐
4. **逐渐加速**：以不出错为原则，稳步提升速度

### 常用左手伴奏型（以 C 大调为例）

**华尔兹（3/4 拍）— Oom-pah-pah：**
```
拍 1：C 根音（Bass Row 第 C 列）
拍 2：C 大三和弦（Major Row 第 C 列）
拍 3：C 大三和弦（Major Row 第 C 列）
```

**进行曲（2/4 拍）— Oom-pah：**
```
拍 1：C 根音
拍 2：C 大三和弦
```

**波尔卡（2/4 拍）：**
```
拍 1 前半：C 根音
拍 1 后半：C 大三和弦
拍 2 前半：G 根音
拍 2 后半：G 大三和弦
```

## 五、入门练习计划

### 第一周目标

- [ ] 右手：C 大调音阶（C D E F G A B C）可以单手匀速演奏
- [ ] 左手：找到 C、F、G 三个根音按钮的位置（凭触觉定位 C 按钮的凸点）
- [ ] 风箱：匀速推拉 4 拍不颤抖

### 第二周目标

- [ ] 右手：能演奏简单五声音阶旋律（如《小星星》）
- [ ] 左手：C 根音 + C 大三和弦交替（进行曲型）
- [ ] 合手：《小星星》慢速合奏

### 推荐入门曲目（由易到难）

1. **《玛丽有只小羊羔》** — 单手右手旋律，4 个音
2. **《小星星》** — 加入简单左手根音配合
3. **《欢乐颂》** — 简单双手合奏
4. **《友谊地久天长》** — 基础华尔兹伴奏型

### 每日练习建议（初学阶段，共 40 分钟）

| 时间 | 内容 |
|------|------|
| 5 分钟 | 热身：单手音阶，左右手分开 |
| 10 分钟 | 技术练习：风箱控制、左手按钮定位 |
| 15 分钟 | 曲目练习：当前练习曲，分手→合手 |
| 10 分钟 | 慢速合手巩固 |

## 六、手风琴类型选择建议

| 你的背景 | 推荐类型 |
|----------|---------|
| 有钢琴基础 | 键盘手风琴（Piano Accordion）|
| 从零开始，目标古典 | 巴扬纽扣手风琴（B 系统或 C 系统）|
| 儿童入门 | 25–34 键键盘手风琴 |
| 民族 / 流行风格 | 键盘手风琴（带斯特拉德拉低音）|
